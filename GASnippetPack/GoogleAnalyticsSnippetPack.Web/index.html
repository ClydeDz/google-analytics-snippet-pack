About

Google Analytics Snippet Pack for Visual Studio helps you write global site tag (gtag.js) code that allows you to send event data to Google Analytics. Due to the nature of this extension, you will find these snippets in HTML and JavaScript Google Analytics Snippet Pack is compatible right from Visual Studio 2012 to 2017 (Community, Professional and Enterprise).

The Google Analytics Snippet extends the code snippet manager of Visual Studio and allows you to insert code snippets into your editor. To insert snippets for the language you're currently writing in your editor, you can use the shortcut keys CTRL+K CTRL+X and then find Google Analytics Snippet as a folder. Alternatively, to bring up the code snippet manager, use the shortcut keys CTRL+K CTRL+B.

How to get it

Download the Google Analytics Snippet from the Visual Studio Marketplace or install directly from Visual Studio by searching for 'Google Analytics Snippet' in the 'Extensions and Updates' window.
